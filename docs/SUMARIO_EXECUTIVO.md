1. Visão Geral do Sistema

O sistema proposto consiste em uma plataforma web com acesso restrito, voltada principalmente para análise e geração de indicadores hospitalares a partir de planilhas.

A solução tem como foco central automatizar o tratamento de dados, reduzindo o esforço manual e acelerando a obtenção de informações relevantes para a gestão hospitalar.

1. Problema Identificado

Atualmente, os dados hospitalares são obtidos de diferentes fontes, como formulários, sistemas médicos e prontuários eletrônicos, sendo posteriormente organizados manualmente em planilhas.

Os principais problemas desse processo são:

- Alto tempo gasto na organização e análise dos dados;
- Processo manual suscetível a erros;
- Dificuldade em transformar dados brutos em indicadores úteis;
- Dependência de uma única pessoa para realizar toda a análise.

Além disso, embora o compartilhamento de dados seja limitado, o principal gargalo está na demora e complexidade da análise dos dados.

1. Objetivo do Sistema

O sistema tem como principal objetivo:

- Automatizar a análise de dados a partir de planilhas;
- Facilitar a criação de indicadores hospitalares;
- Reduzir significativamente o trabalho manual;
- Aumentar a velocidade na geração de informações estratégicas.

Como objetivo secundário, o sistema também permitirá a visualização e o compartilhamento dos indicadores entre diferentes setores.

1. Público-Alvo

O sistema será utilizado por:

- Coordenadores de setores (ex: UTI);
- Gestores hospitalares;
- Profissionais responsáveis pela análise de dados.
1. Funcionalidades Principais

O sistema contará com as seguintes funcionalidades:

- Acesso restrito por login;
- Upload de planilhas;
- Processamento automático dos dados enviados;
- Geração automática de indicadores e métricas;
- Visualização de gráficos e relatórios;
- Organização dos dados por período (mensal, trimestral, etc.);
- Navegação por setores hospitalares.
1. Benefícios Esperados

Com a implementação do sistema, espera-se:

- Redução significativa do tempo de análise de dados;
- Diminuição de erros manuais;
- Padronização na geração de indicadores;
- Maior agilidade na tomada de decisão;
- Redução da dependência de processos manuais;
- Base estruturada para futuras automatizações
